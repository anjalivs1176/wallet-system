# 💳 Wallet System - Microservices Backend

## 🧠 Architecture

This project follows a **microservices architecture**:

* 🔐 **User Service**

  * User registration & login
  * JWT authentication

* 💰 **Wallet Service**

  * Wallet creation
  * Balance management
  * Money transfer
  * Transaction history

---

## 🚀 Tech Stack

* Java
* Spring Boot
* Spring Security
* JWT (Authentication)
* MySQL
* Docker (for multi-service setup)

---

## ✨ Features

* Secure user authentication using JWT
* Wallet-to-wallet money transfer
* Transaction history tracking
* Global exception handling
* Dockerized services

---

## 📦 Project Structure

wallet-system/
├── user_service/
├── wallet_service/
├── docker-compose.yml

---

## ▶️ How to Run Locally

### 1. Clone the repository

git clone https://github.com/anjalivs1176/wallet-system.git

### 2. Navigate to project

cd wallet-system

### 3. Run using Docker

docker-compose up
